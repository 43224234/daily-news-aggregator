# Daily News Aggregator

A personal news aggregator that collects news from BBC, CNBC, TechCrunch, Azure Blog, and Google AI.

## Features

- **Daily News Collection**: Automatically fetches news from multiple sources.
- **Automatic Translation**: Headlines and summaries are translated into Chinese.
- **Categories**: International, Business, Sport, Stocks, and AI.
- **Save for Later**: Save interesting articles to your local browser storage.

## Live Website

[**Visit the Daily News Aggregator**](https://43224234.github.io/daily-news-aggregator/)

## How to Run Locally

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000/daily-news-aggregator](http://localhost:3000/daily-news-aggregator) in your browser.

3.  **Fetch News**:
    ```bash
    node scripts/fetch-news.js
    ```
